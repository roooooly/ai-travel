# AI旅行家 - 完整产品方案

> Version: 0.2
> 核心定位: AI驱动的一站式旅行规划平台

---

## 一、产品slogan与愿景

### Slogan
**"你的专属旅行管家"**

或者更有记忆点：
**"跟着AI，去哪儿都行"**

### 核心体验
```
输入: "苏州3天" + 热点关键词
     ↓
AI分析热点 → 匹配路线 → 输出可编辑行程
     ↓
用户选择/调整 → 确定行程 → 一键导航
```

| 文件存储 | 阿里云OSS | 图片/文件 |

### 5.3 数据库设计

```sql
-- 目的地表
CREATE TABLE destinations (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),           -- 苏州
    name_en VARCHAR(100),
    country VARCHAR(50),         -- 中国
    region VARCHAR(50),          -- 江苏
    description TEXT,
    best_season VARCHAR(100),    -- 最佳旅游季节
    tags TEXT[],                  -- ['园林', '古镇', '美食']
    hot_level INT DEFAULT 0,      -- 热度指数
    created_at TIMESTAMP
);

-- 景点表
CREATE TABLE attractions (
    id SERIAL PRIMARY KEY,
    destination_id INT REFERENCES destinations(id),
    name VARCHAR(200),
    category VARCHAR(50),        -- 园林/古镇/博物馆/自然
    address VARCHAR(300),
    latlng JSONB,                -- {"lat": 31.3, "lng": 120.6}
    ticket_price DECIMAL(10,2),
    opening_hours VARCHAR(100),
    recommended_duration INT,      -- 分钟
    parking_info JSONB,          -- [{"name": "xxx", "price": 10}]
    photo_spots JSONB,           -- [{"name": "xxx", "tips": "xxx"}]
    tags TEXT[],
    rating DECIMAL(2,1),
    local_tips TEXT[],            -- 本地人才知道的tips
    created_at TIMESTAMP
);

-- 美食表
CREATE TABLE restaurants (
    id SERIAL PRIMARY KEY,
    destination_id INT REFERENCES destinations(id),
    name VARCHAR(200),
    category VARCHAR(50),         -- 苏帮菜/小吃/网红店
    address VARCHAR(300),
    latlng JSONB,
    avg_price DECIMAL(10,2),
    tags TEXT[],
    recommended_dishes TEXT[],
    business_hours VARCHAR(100),
    parking_info VARCHAR(200),
    rating DECIMAL(2,1),
    local_review TEXT,            -- 本地人评价(区别于大众点评)
    photo_spots JSONB,           -- 出片机位
    created_at TIMESTAMP
);

-- 打卡点表
CREATE TABLE checkin_spots (
    id SERIAL PRIMARY KEY,
    attraction_id INT REFERENCES attractions(id),
    name VARCHAR(200),
    description TEXT,
    photo_url VARCHAR(500),
    user_tips TEXT,
    created_at TIMESTAMP
);

-- 用户表
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    nickname VARCHAR(100),
    avatar_url VARCHAR(500),
    preferences JSONB,           -- {"food": ["辣"], "transport": "地铁"}
    checkin_count INT DEFAULT 0,
    created_at TIMESTAMP
);

-- 行程表
CREATE TABLE itineraries (
    id SERIAL PRIMARY KEY,
    user_id INT REFERENCES users(id),
    destination VARCHAR(100),
    days INT,
    title VARCHAR(200),
    content JSONB,              -- AI生成的行程详情
    status VARCHAR(20),         -- draft/published
    view_count INT DEFAULT 0,
    like_count INT DEFAULT 0,
    created_at TIMESTAMP
);

-- 社区帖子表
CREATE TABLE posts (
    id SERIAL PRIMARY KEY,
    user_id INT REFERENCES users(id),
    destination VARCHAR(100),
    title VARCHAR(200),
    content TEXT,
    images TEXT[],
    likes INT DEFAULT 0,
    comments INT DEFAULT 0,
    tags TEXT[],
    created_at TIMESTAMP
);
```
